# Videos Ordner

Legen Sie hier Ihre Video- und Bilddateien ab.

## Benötigte Dateien:

### Videos (.mp4):
- `day7.mp4` - Tag 7: Ein lustiges Weihnachtsvideo
- `day16.mp4` - Tag 16: Ein herzerwärmendes Weihnachtsvideo
- `day20.mp4` - Tag 20: Nur noch 4 Tage bis Heiligabend
- `day24.mp4` - Tag 24: FROHE WEIHNACHTEN!

### Bilder (.jpg oder .png):
- `day13.jpg` - Tag 13: Magische Winterstimmung
- `day22.jpg` - Tag 22: Fast geschafft! Bald ist Weihnachten

## Format-Empfehlungen:

**Videos:**
- Format: MP4 (H.264)
- Auflösung: 1280x720 oder 1920x1080
- Dateigröße: < 50 MB pro Video (für bessere Performance)
- Länge: 30 Sekunden bis 2 Minuten

**Bilder:**
- Format: JPG oder PNG
- Auflösung: 1920x1080 oder 1280x720
- Dateigröße: < 5 MB pro Bild

## Verwendung:

Die Dateien werden automatisch über den Server bereitgestellt:
- Videos: `http://localhost:3000/videos/day7.mp4`
- Bilder: `http://localhost:3000/videos/day13.jpg`

Einfach die Dateien in diesen Ordner kopieren und der Adventskalender kann sie direkt verwenden!
